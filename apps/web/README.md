# Conn web

Vercel-ready React + Vite landing page for Conn, built as a workspace inside the native app repository.

```bash
pnpm install
pnpm web:dev
```

For Vercel, import the repository and set the Root Directory to `apps/web`. The framework preset is detected as Vite.

The interactive product demo is self-contained under `src/demo/`. The original
root-level design playground and its development notes are intentionally not
part of the public repository.
