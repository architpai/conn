# Conn is the product name

Status: Accepted
Date: 2026-07-19

The product formerly called Sidequest is named Conn. “Conn” captures the product's temporary steer, approve, and interrupt authority over Codex-owned work without implying ownership; “Backseat” was rejected because it suggests unsolicited interference and collides with the existing [BackseatAI](https://www.backseat.cloud/) developer product.

Active app branding, Swift modules, executables, tests, and canonical documentation use `Conn` or `conn`. Historical phase evidence, archived plans, captured fixtures, and provenance retain the name Sidequest. The legacy `dev.sidequest.app` bundle identifier, `Application Support/Sidequest` storage roots, `sidequest-hook-event-v1` identity namespace, and every installed-plugin machine or path identity—including the `sidequest` selectors, marketplace and plugin paths, relay executable, and management script—remain stable until a deliberate migration or the Phase 11 plugin retirement prevents preference loss, event duplication, broken hooks, and orphaned plugin installs.
